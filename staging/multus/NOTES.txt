4. To use Multus for multi-interface pods, create NetworkAttachmentDefinition resources:

   Example:
   ```yaml
   apiVersion: "k8s.cni.cncf.io/v1"
   kind: NetworkAttachmentDefinition
   metadata:
     name: macvlan-conf
   spec:
     config: '{
       "cniVersion": "0.3.1",
       "type": "macvlan",
       "master": "eth0",
       "mode": "bridge",
       "ipam": {
         "type": "host-local",
         "subnet": "192.168.1.0/24"
       }
     }'
   ```

5. Then annotate your pods to use additional networks:
   ```yaml
   metadata:
     annotations:
       k8s.v1.cni.cncf.io/networks: macvlan-conf
   ```

6. For more information, visit: https://github.com/k8snetworkplumbingwg/multus-cni
